Sobre o projeto

Este projeto que tem como objetivo contribuir para a otimização do uso dos aparelhos das academias ao ar livre, utilizando um sistema web para o registo de atividades realizadas nessas áreas, na cidade de Várzea Paulista. Através de comunicação com o Ginásio de Esporte da prefeitura, professores e potenciais usuários das academias foi possível identificar, compreender e desenvolver uma solução tecnológica adequada que aproxime usuários e professores voluntários para uma melhor utilização desses espaços públicos. Realizando pesquisas bibliográficas e utilizando do aprendizado em cursos já realizados na UNIVESP como Fundamentos da Internet e Web, Estrutura de Dados e outros, foi escolhido uma linguagem de programação, um framework web que utiliza noções de banco de dados praticando controle de versão. O desenvolvimento de um sistema web envolveu a utilização de diversos programas e tecnologias. Inicialmente, foi escolhida a linguagem de programação Python como base para o projeto, foi adotado o framework Django, já para a parte front-end, foram utilizadas linguagens como HTML e CSS para a criação da interface do usuário, garantindo uma experiência agradável e simples. O controle de versão do código foi feito através do GitHub. Por fim, o banco de dados SQLite foi escolhido para armazenar os dados do sistema de forma simples e eficiente durante o desenvolvimento. Com o sistema web, pretende-se beneficiar a população local ao otimizar o uso do espaço público aproximando usuários e profissionais da área de Educação Física.
